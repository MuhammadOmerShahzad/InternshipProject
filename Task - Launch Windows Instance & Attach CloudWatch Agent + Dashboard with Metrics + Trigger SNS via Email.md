# Task: Launch Windows Instance & Attach CloudWatch Agent + Dashboard with Metrics + Trigger SNS via Email

- A **Windows EC2 Instance** was created manually via the AWS Management Console.
- Accessed the instance securely using the **Remote Desktop Protocol (RDP)**.
- Installed and configured the **CloudWatch Agent** via AWS Systems Manager directly inside the instance.
- Ran the CloudWatch Agent to monitor system metrics.
- Attached CloudWatch metrics to a Dashboard to actively track the **CPU Utilization**.
- Configured an **SNS (Simple Notification Service)** topic to trigger and send email notifications based on the CPU Utilization metrics.
